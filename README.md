# Shop-Bot
# 🛒 ShopBot — Telegram-витрина магазина

Бот на **aiogram 3.x** + **SQLite**, готовый к деплою на **Render**.

---

## 🗂 Структура проекта
shop-bot-demo/
├── main.py # точка входа (Flask + aiogram)
├── database.py # вся работа с SQLite
├── keyboards.py # клавиатуры и форматирование
├── requirements.txt
├── .python-version # фиксация версии Python 3.13
└── handlers/
├── init.py
├── client.py # команды клиента
└── admin.py # команды администратора (демо-режим)

text

## ⚙️ Переменные окружения

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |

> **ADMIN_ID не требуется** — админ-панель работает в демо-режиме для всех.

---

## 🚀 Локальный запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Задать переменные окружения
export BOT_TOKEN="your_token_here"

# 3. Запустить
python main.py
☁️ Деплой на Render
Создайте Web Service (не Background Worker, чтобы работал Flask).

Укажите Start Command: python main.py

В разделе Environment Variables добавьте BOT_TOKEN.

Render сам установит порт 10000 через переменную PORT.

🤖 Команды бота
Клиент
Кнопка	Действие
Каталог	Просмотр всех товаров с фото, ценами и описанием
Корзина	Просмотр добавленных товаров и итоговой суммы
Оформить заказ	Оформление заказа (демо-режим)
Контакты	Ссылка на разработчика
Администратор (Админ-панель → /add_product)
Команда	Действие
/add_product	Пошаговое добавление товара (название → описание → цена → фото)
⚠️ Демо-режим: товары не сохраняются в базу данных. Это витрина для демонстрации функционала.

✅ Возможности
Каталог товаров с фото, ценами и описанием

Корзина с автоматическим подсчётом итога

Оформление заказа в один клик

Админ-панель для добавления товаров

Flask keep-alive предотвращает засыпание сервиса на Render
